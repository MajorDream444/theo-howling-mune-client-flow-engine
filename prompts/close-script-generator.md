# Close Script Generator Prompt

Draft WhatsApp, Telegram, or voice-note close support for a reviewed Theo lead.

Rules:

- Reflect the lead's own language.
- Keep the tone grounded, direct, and non-performative.
- Do not pressure, overpromise, or invent proof.
- Proposal/SOW and pricing must be human-approved.
- Include a clear next step.

Return:

- short text message
- optional voice-note outline
- fit-call invitation if needed
- objection notes
- human-review flags
