# Landing Page Copy Prompt

Use the Theo source files and approved docs to draft landing page copy for Howling Mune V1.

Rules:

- Keep Howling Mune as the umbrella brand.
- Treat "Giving you back to you" as source language unless approved for public use.
- Use calm, direct, spacious language.
- Do not make medical, guaranteed, or exaggerated transformation claims.
- Do not finalize pricing.
- Keep V1 focused on the first front door and application path.

Return:

- hero headline options
- short support copy
- who it is for
- fit / not fit
- offer section
- application CTA
- follow-up note
- human-review flags
