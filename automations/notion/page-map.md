# Notion Page Map

Notion is the client-facing experience layer.

## Theo Dossier Page

Purpose: private operator/client dossier for Theo V1.

Recommended sections:

- case summary
- source inventory
- founder profile
- doctrine map
- ICP
- offer architecture
- system map
- launch checklist
- human-review decisions

## Source Page

Purpose: preserve source material links and ingestion status.

Include:

- source title
- source type
- date
- canonical/supporting/superseded status
- file path or attachment link
- extraction notes

## Blueprint Page

Purpose: reviewed blueprint output.

Include:

- founder profile
- doctrine
- offer path
- ICP
- route map
- build tasks
- proposal readiness
- review flags

## Public / Client-Facing Page

Only use if approved. This can become the front-door page or a private pre-launch review page. Public claims and copy require human approval before publishing.
